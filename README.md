#### 关键特性

* 工作电压范围：2.2V ~ 5.5V
* 封装：TSSOP/SOP20，SOP16
* 8K Word FLASH程序存储器
* 1K Word Data FLASH
* 1KBytes SRAM
* 支持ISP，IAP，ICD功能
* 一个8位定时器
* 两个12位定时器
* 一个16位的多功能定时/计数器
* PWM输出占空比可独立设置
* PWM输出极性可独立设置
* 支持7种工作模式：定时器模式，捕捉模式，比较模式，PWM模式，单脉冲模式，关断功能模式，从模式
* 两路高速异步收发器UART0/UART1
* 一路同步串口通讯模块SPI,一路I2C总线
* ADC：12位精度，12+1通道
